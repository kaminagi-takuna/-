# AIアシスタント用プロジェクト引き継ぎ資料 (AI Handover Document)

このドキュメントは、「神薙拓那（なぎぃ）オンラインレッスン」のホームページ構築プロジェクトの現状と仕様を、他のAIアシスタントに引き継ぐためのものです。

## 1. プロジェクト概要
- **目的**: ボイストレーナー・声優講師である神薙拓那さんの個人ホームページ・レッスン案内サイトの作成。
- **現在の状況**: 初期構築からモバイル端末へのレスポンシブ対応（ハンバーガーメニューや改行位置のスマホ最適化）が完了しています。Vercel等での自動デプロイを想定し、ビルドエラーを回避するためのLintエラー修正も完了済みのクリーンな状態です。
- **プロジェクトパス**: `c:/Users/grste/.gemini/antigravity/scratch/homepage`

## 2. 技術スタック
- **環境構築**: Vite
- **フレームワーク**: React (v19)
- **パッケージマネージャ**: npm
- **スタイリング**: Vanilla CSS (`App.css`, `index.css`)
  - ※Tailwind CSS等は使用しておらず、CSS変数と純粋なCSSクラスでデザインシステムを構築。

## 3. 主要ファイルの役割
- `src/App.jsx`: SPA（単一ページアプリケーション）として機能するメインコンポーネント。
  - `activeTab` というStateを用いて、疑似的にページ（タブ）の切り替えを行っています。
  - `isMobileMenuOpen` Stateを用いてモバイル画面時のメニュー開閉を管理しています。
- `src/App.css`: 全体のデザイン。
  - 後半に `@media screen and (max-width: 768px)` のメディアクエリがあり、モバイル表示用の詳細なスタイル定義があります。スマホ専用の改行要素として `.sp-only` クラスが機能しています。

## 4. 実装済みのコンテンツ（タブ）
1. **ようこそ (`welcome`)**: ウェブサイトのトップページ。
2. **自己紹介 (`profile`)**: 経歴およびパーソナルデータ（好きなもの等）。
3. **レッスン内容及び金額表 (`lessons`)**: 昼間・夜間・プロコースなどの料金体系をカード型で表示。
4. **依頼詳細 (`canva`)**: 原稿作成などの各種依頼に関する詳細、文例集（アコーディオン表示 `details`・`summary` タグを使用）。
5. **Twitter (`twitter`)**: 𝕏の指定ツイートをWidgetスクリプトを利用して自動埋め込み表示。
6. **note (`note`)**: noteの記事を `iframe` で埋め込み表示。
7. **お問い合わせ (`contact`)**: JotFormを利用したお問い合わせ・申し込みフォームの `iframe` 埋め込み。

## 5. UI/UX設計のポイント
- **ダークテーマ＆グラスモーフィズム基調**: `var(--sidebar-bg)` などの環境に合わせた透明感のあるダークテーマ。
- **レスポンシブデザイン**:
  - PCでは左に固定サイドバー、右側にコンテンツの構成（Flexbox）。
  - モバイル（768px以下）では上部に固定ヘッダー（ハンバーガーメニューあり）が表示され、サイドバーはドロワーとして左からスライド。
- カスタムアニメーション (`fade-in`) による、タブ遷移時の滑らかな描画。

## 6. 次のAIアシスタントへの指示・お願い
- 本プロジェクトを再開する際は、ターミナルで `cd c:/Users/grste/.gemini/antigravity/scratch/homepage` に移動し、`npm run dev` でローカルサーバーを立ててください。
- 新しいページやコンテンツを追加する場合は、`src/App.jsx` 内の `renderContent` 関数における `switch` 文に新ケースを追加し、サイドバー（`nav-menu`）にメニュー項目を追加してください。
- 編集を行う際は、現在のクリーンなデザイン規約（クラス・CSS変数の再利用）を守って構築を進めてください。
